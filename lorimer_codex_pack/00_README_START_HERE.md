## How to use this pack
1. Put all markdown files in the repo root or a `/docs/lorimer/` folder.
2. Add `PROJECT.md` to the repo root if your Codex workflow expects it there.
3. Attach or place the PDF in the repo or upload it into the Codex workspace.
4. Start with the prompt in `13_COPY_PASTE_PROMPTS.md` under `MASTER ORCHESTRATION PROMPT`.
5. Have Codex implement in phases using `12_BUILD_SEQUENCE.md`.
6. Run QA against `11_QA_ACCEPTANCE_CRITERIA.md`.

## Locked direction
- ultra minimal
- strict editorial asymmetry
- black text on white background
- quiet luxury
- editorial ecommerce
- mobile and responsiveness are top priority
- spacing system is the primary quality signal
- Shopify integration later, not now

## Important
This is not a normal ecommerce site and should not drift toward default storefront patterns.
The coded result should feel like a lookbook and archive that also sells products.
