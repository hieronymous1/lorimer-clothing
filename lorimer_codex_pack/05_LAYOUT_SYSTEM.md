# 05_LAYOUT_SYSTEM

## Core Rule
Asymmetry must be systematic.
Do not improvise compositions across the site.

## Approved Desktop Layout Patterns

### Pattern A — balanced split
- 6 columns / 6 columns
Use for:
- balanced image pairings
- symmetrical breathing moments
- calmer sections

### Pattern B — dominant left
- 8 columns image / 4 columns text or support
Use for:
- product detail
- editorial features
- collection statements

### Pattern C — dominant right
- 4 columns text or support / 8 columns image
Use for:
- alternating rhythm
- lookbook variation
- offset pacing

Only use these patterns unless there is a compelling reason otherwise.

## Home Page Structure
1. Logo block
2. Inline nav under logo
3. Hero / rotating images
4. Products preview using editorial rows
5. S/S24 collection preview
6. Footer

### Home Page behavior rules
- toolbar disappears on scroll
- inline nav becomes primary
- home page S/S24 uses group image and collection level label, not Look 1
- products preview should feel like an index, not a mini storefront grid

## Products Page Structure
1. Logo and category block
2. Product listings
3. Divider / horizontal editorial image
4. S/S24 related subset
5. Footer

### Products Page behavior rules
- no top toolbar
- categories should behave like quiet editorial navigation, not chips
- divider images must be larger and meaningful
- every product needs its own destination page

## S/S24 Page Structure
1. Intro / logo block
2. Alternating look blocks
3. More images per look navigation
4. Final wide image
5. Footer

### S/S24 Page behavior rules
- reduce unnecessary vertical distance between looks
- images should dominate more
- text should be smaller, cleaner, and carefully placed
- "View in Products" must link into the relevant part of Products

## Product Detail Page Structure
Desktop:
- left text, right image dominant

Mobile:
- image stack first, text below

Behavior rules:
- product should feel like an object study
- avoid text heavy layout
- product imagery must dominate the experience more than currently
