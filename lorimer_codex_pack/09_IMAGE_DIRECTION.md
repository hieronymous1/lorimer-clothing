# 09_IMAGE_DIRECTION

## Why image direction matters
Minimal design exposes inconsistency.
If the image system is weak, the site immediately looks unfinished.

## Current image mix observed
- product cutouts on white
- runway / event images
- outdoor fashion images
- horizontal editorial separators

This can work only if each image type is assigned a clear role.

## Recommended image roles

### Product cutouts
Use for:
- clean catalogue moments
- direct object focus
- clarity in product browsing

### Environmental / runway images
Use for:
- editorial interruptions
- S/S24 narrative
- collection framing

### Wide horizontal images
Use for:
- section transitions
- pacing resets
- breathing moments

## Rules
- maintain consistent ratio per content type
- ensure hover images match the specific product
- make final image selection intentional across Home, Products, S/S24, and product pages
- avoid random source mixing with no compositional logic

## Required content task
Select all final images used for:
- Home Page
- Products
- S/S24
- all individual product pages
