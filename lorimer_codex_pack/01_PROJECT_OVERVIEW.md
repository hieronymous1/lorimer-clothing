# 01_PROJECT_OVERVIEW

## What this project is
Lorimer is a luxury minimal editorial ecommerce site for a clothing brand.
The site should feel museum-like, sparse, controlled, and expensive through restraint.

## What we are building
We are not building a normal online store.
We are building a fashion archive, lookbook, and catalogue system that also supports product browsing.

## Key perception goals
The user should feel:
- calm
- precision
- spaciousness
- restraint
- hierarchy
- confidence

## Key anti goals
The user should not feel:
- crowding
- generic ecommerce
- loud luxury
- template repetition
- UI clutter
- over animation

## Strategic conclusion
The Framer site is close visually, but it still behaves too much like a layout and not enough like a system.
This project must be translated into:
- a spacing system
- a layout system
- a mobile recomposition system
- a reusable component system
- a QA system
